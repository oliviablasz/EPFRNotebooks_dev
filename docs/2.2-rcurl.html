2.2 RCurl

In this demonstration we will use the function ftp.get() from library('EPFR) to access files from EPFR’s ftp which requires the RCurl library.

library(RCurl)

To use this function you will set a file path to the csv file that will be used for your strategy and a local folder that this file will be downloaded to:

file <- "EPFRHistory/Strategies/daily/FloPctCtry-daily.csv" # FTP FILE - FloPctCtry-daily.csv
fldr <- "C:\\temp" # LOCAL FOLDER

Next you will enter your username and password:

usr <- "YOUR USER NAME" # FTP USERNAME
pwd <- "YOUR PASSWORD" #FTP PASSWORD

Now you can call ftp.get() with the following inputs:

ftp.get(file, fldr, "filetransport.epfrglobal.com", usr, pwd, "ftp") # DOWNLOADS FILE FROM FTP TO LOCAL FOLDER, USING CREDENTIALS

After the file is downloaded the flow file’s local path can be stored with follow line:

flow.file <- paste(fldr, ftp.file(file), sep = "\\") # STORES FLOW FILE PATH
